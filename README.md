## MikhmonPlus PPPoe (https://github.com/laksa19/mikhmonv3)
Arch Support : linux/amd64, linux/arm64,

```sh
$ docker run --name mikhmonplus -p <YOUR-PORT>:80 riadag/mikhmonplus:latest
```

Example 
```sh
$ docker run --name riadag/mikhmonplus -p 8082:80 riadag/mikhmonplus:latest
```

Mikhmon running on 8082 port on your machine host.
